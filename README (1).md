# 🏠 House Price Prediction using Linear Regression

## 📌 Internship Task 1 – Machine Learning with Python

This project is part of my Machine Learning internship where I built and evaluated a Linear Regression model using the California Housing Dataset. The goal is to predict house prices based on different housing features.

---

## 🎯 Objective

* Build a Linear Regression model
* Predict house prices using housing-related features
* Evaluate model performance using regression metrics
* Understand the complete Machine Learning workflow

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook / VS Code

---

## 📂 Dataset

The project uses the California Housing Dataset available in Scikit-Learn.

### Features:

* Median Income (MedInc)
* House Age (HouseAge)
* Average Rooms (AveRooms)
* Average Bedrooms (AveBedrms)
* Population
* Average Occupancy (AveOccup)
* Latitude
* Longitude

### Target Variable:

* Median House Value (MedHouseVal)

---

* Data Loading and Exploration
* Missing Value Analysis

* Correlation Analysis
* Data Visualization

* Linear Regression Model Training
* Model Evaluation

## 📈 Visualizations

## 🔍 Model Evaluation Metrics

The model was evaluated using:

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

---

## 🚀 Key Insights

* Median Income has the strongest impact on house prices.
* House prices vary significantly across different locations.
* Linear Regression provides a good baseline model for prediction.
* Model performance can be improved using advanced regression techniques.

## 🚀 How to Run

1. Clone this repository

```bash
git clone <repository-link>
```

2. Create Virtual Environment

```bash
python -m venv venv
```

3. Activate Virtual Environment

```bash
venv\Scripts\activate
```

4. Install Required Libraries

```bash
pip install pandas numpy scikit-learn matplotlib seaborn jupyter
```

5. Open Jupyter Notebook

```bash
jupyter notebook
```

6. Run all cells in the notebook
